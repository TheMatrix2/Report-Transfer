# Report transfer from Yandex Metrika to MariaDB
## Setting up (Linux):

```shell
git clone https://github.com/TheMatrix2/Report-Transfer
```
```shell
cd Report-transfer
```
```shell
cp .env.example .env
```
```shell
chmod +x setup.sh
```
```shell
./setup.sh
```
```shell
chmod +x schedule.sh
```
```shell
./schedule.sh
```
